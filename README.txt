EduVision AI - Giải bài tập bằng hình ảnh và AI

Mô tả:
- Đây là demo web cho phép tải ảnh bài tập toán, mô phỏng xử lý OCR và giải bài toán bằng AI.
- Có chatbot đơn giản giúp hỏi đáp cùng AI.
- Có form đăng nhập giả lập (chưa tích hợp backend).

Hướng dẫn chạy:
- Mở file index.html trên trình duyệt hỗ trợ (Chrome, Firefox,...)
- Tải ảnh bài tập, bấm "Xem lời giải" để xem lời giải mẫu.
- Chat với AI bằng khung chat ở phần Chatbot.
- Đăng nhập bằng form ở phần Đăng nhập (chỉ demo).

Lưu ý:
- Đây là bản demo minh họa, bạn cần tích hợp API AI thực tế để có kết quả chính xác.
- Bạn có thể phát triển thêm backend và AI theo ý muốn.
